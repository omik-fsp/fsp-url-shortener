Url Shortener Microservice

Porting FreeCodeCamp APIs and Microservices Certification Challenges from JS to Python

  • I can POST a URL to /api/shorturl/new and I will receive a shortened URL in the JSON response. e.g. : {"original_url":"http://omik.github.io","short_url":499dE5}
  • If I pass an invalid URL that doesn't follow the valid http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"Not a valid URL"}.
  • When I visit the shortened URL, it will redirect me to my original link.
    e.g. /api/shorturl/TQfpV3
